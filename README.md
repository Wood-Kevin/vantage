# Vantage

25+ tools for life's quick questions.

A cross-platform mobile utility app for iOS and Android built with Expo, React Native, and TypeScript.

## Stack
- Expo (Managed Workflow)
- React Native + TypeScript
- Expo Router (file-based navigation)
- NativeWind v4 (Tailwind CSS)
- RevenueCat (in-app purchases)
- Expo SQLite (local storage)

## Development
Built and tested on Linux VM via VS Code Remote SSH.
Testing on physical Android device via Expo Go.
